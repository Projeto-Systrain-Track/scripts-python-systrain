# 📊 Monitoramento de Recursos do Sistema com Python

Este projeto consiste em um script em Python para **captura e registro contínuo de métricas do sistema**, como uso de CPU, memória RAM e atividade de disco.

Os dados coletados são armazenados em um arquivo `.csv`, podendo ser utilizados posteriormente para **análise, dashboards ou sistemas de monitoramento**.

---

## 🚀 Funcionalidades

- 📈 Monitoramento de uso de CPU (%)
- 💾 Monitoramento de memória RAM (uso em bytes)
- 📀 Cálculo de atividade de disco (throughput aproximado)
- 🕒 Registro de data e hora de cada coleta
- 📁 Exportação automática para arquivo CSV
- 🔄 Coleta contínua em tempo real

---

## 🛠 Tecnologias utilizadas

- Python 3
- psutil (coleta de métricas do sistema)
- csv (armazenamento dos dados)
